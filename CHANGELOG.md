# 更新日志

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [0.3.0] - 2026-01-01

### 新增
- 新增多种 RAG 研究模块及评估工具页面
- 添加所有向量数据库列表展示
- 添加 embedding 模型模块及相关页面
- 添加 RAGAS 评估工具页面和评估工具对比功能
- 添加向量数据库对比页面及组件

### 变更
- 更新页面标题和描述中的术语
- 将"Agent"统一修改为"智能体"

## [0.2.2] - 2026-01-01

### 新增
- 添加导航栏，统一页面布局样式
- 添加 RAG、向量数据库和知识图谱子模块

### 变更
- 统一页面主容器和主内容区域宽度样式
- 调整网格布局和卡片样式
- 简化页脚样式并移除多余文本
- 重构技术栈模块结构，将数据分散到各页面目录
- 重构页面导航结构并移除无用代码

## [0.2.1] - 2026-01-01

### 变更
- 重构组件数据为模块化堆栈数据结构
- 统一页面的导入格式并简化 Footer 组件

## [0.2.0] - 2026-01-01

### 新增
- 添加组件详情页面，展示各个 Agent 技术组件的详细信息
- 添加应用版本和构建日期显示功能
- 提取 Footer 为独立组件，增加版本显示功能
- 添加组件数据结构，支持组件信息的结构化管理

### 变更
- 使用 slug 替代数字作为组件路由标识，提升 URL 可读性
- 使用 Link 组件替代 a 标签以优化导航性能
- 简化组件链接路径，优化页面跳转体验
- 统一代码格式和引号使用规范

### 持续集成
- 修改部署触发条件为标签推送并修复缩进

## [0.1.0] - 2026-01-01

### 新增
- 重新设计网站首页，展示 Agent 技术栈的核心组成部分
- 应用 Swiss 设计风格，包括网格系统、左对齐排版、高对比度配色
- 添加 8 个 Agent 技术核心组件介绍（大语言模型、提示工程、工具调用、记忆系统、规划与推理、多模态能力、自主性、协作能力）
- 更新 CSS 变量系统，支持 Swiss 设计规范（字体大小、间距、颜色、行高）
- 支持暗色模式适配

### 变更
- 更新页面元数据（标题、描述）

## [0.0.1] - 2025-11-17

初始化项目模板。
